# KEHOMGE

Multiple Equilibria in a simple GE Model (KEHOMGE, SEQ=149).

This example mirrors the GAMS MPSGE model and demonstrates:
- a clean MPSGE block file in `kehomge.mpsge`
- an MPSGE.jl translation in `mpsge_model.jl`
- a block-based MCP model in `KEHOMGE.jl` derived from the MPSGE structure
- equation listings in `equations.md`

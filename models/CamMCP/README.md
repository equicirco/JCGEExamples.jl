# CamMCP

Cameroon CGE model reformulated as an MCP (CAMMCP, SEQ=129), built entirely from reusable blocks and solved via PATHSolver.

Reference:
Condon, T, Dahl, H, and Devarajan, S. Implementing a Computable General Equilibrium Model on GAMS - The Cameroon Model. The World Bank, 1987.

Equation dump is saved in `packages/JCGEExamples/models/CamMCP/equations.md`.
